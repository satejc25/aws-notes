Q)What is VPC ?
VPC is Virtual Private Cloud which helps us to create our own network as it improves security and keeps our server safe.

Q)What are the Subnets when working with VPC ?
Subnets are range of ip addresses which are used to assign private ip's to our server. By divinding larger range ip,s to smaller groups are called as subnetting.Subnetting give the count of how much machine we can launch from this ip range. 

Q)What is NAT Device?
NAT is Network Address Translation.NAT devices are used for private subnets where private subnet has no access ti internet. When we apply nat gateway with the help of public sever we can access our private server.
